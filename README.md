# Studious-study-prototype

Studious study is a mobile application prototype designed to help students improve their study habits by organizing study sessions, tracking progress, and setting achieveable study goals. The app focuses on creating a simple, intuitive and motivational experience that encourages confidence with consistent learning and engagement. 


The purpose of this project was to design a user-friendly application that demonstrates effective mobile app design, principles, user interaction, navigation and data management. This project represents the planning and design phase of the application.



My target audience are Middle school students, High school students, college students and adult learners looking to improve their study habits.

Core Feautures of the app are start and manage study sessions, track completed study sessions, monitoring study progress, set study goals and personalize the app with user settings.


Studious study was designed with a clean and intuitive interface that allows users to navigate easily between screens. Simple layouts, clear buttons, and consistent icons helped to create an enjoyable user experience.
